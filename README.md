# VRCMT-stub

Reemplazo del `yt-dlp.exe` de VRChat para cookies, yt-dlp actualizado y filtrado de flags legacy. Usado por [VRCMT](https://github.com/doctorlatex/VRCMT).

## Configuración

`%LOCALAPPDATA%\VRCMT\vrchat_stub.json` (VRCMT lo escribe desde Ajustes).
